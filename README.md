# Google Maps Search App

This project provides a simple Flask application that searches Google Places in multiple cities for multiple keywords and displays the results on an interactive Google Map.

## Setup

1. Install Python dependencies:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
python app.py
```

The server will be available at `http://localhost:5000/`.

## Configuration

The Google API key is stored in `config.py` as `GOOGLE_API_KEY`. Update it with your own key if needed.
